# Todo API

A simple Todo API built with Node.js, Express.js, MongoDB Atlas, and Mongoose.

## Features

- Add Todo
- Get All Todos
- Update Todo
- Delete Todo
- Mark Todo as Complete

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Run Locally

```bash
npm install
node index.js
```

Server runs on:

```bash
http://localhost:3000
```
